Sublime Text3 安装主题与背景

关于Sublime Text 3安装主题与背景的方法

第一次使用MacBook 的TIME MECHINE，把系统恢复了一下，没有想到把我电脑上Sublime Text3 的一些个性化设置也“恢复出场设置了”，所以我打算重新来一遍个性化的定制，并且将其详细步骤记录下来，已备日后查看。

安装主题

Step 1 安装包控制

同时按住电脑按键cmd + shift + p，在打开的控制台中输入pack control，用光标选择对应项，回车安装。

安装ayu主题

ayu 主题是我重置电脑前用的主题，现在我将它重新安装。

同时按住电脑按键cmd + shift + p，在打开的控制台中输入Install Package并回车确认，稍等片刻后，输入ayu，回车确认，在安装过程中，会提示是否安装file icon，这个是用于文件的图标显示，也把它安装了，点击Install。

设置主题

安装完毕后，在屏幕左上角，选择Sunlime Text -> Preferences -> Settings，打开设置页面。

字体

介绍

这是一个专为程序员写程序设计的字体 —— Fira Code
Fira 是 Mozilla 公司（火狐浏览器她爹）主推的字体系列。Fira Code 是其中的一员，专为写程序而生。出来具有等宽等基本属性外，还加入了编程连字特性（ligatures）。
Fira Code 就是利用这个特性对编程中的常用符号进行优化，比如把输入的!=直接显示成≠或者把>=变成≥等等，以此来提高代码的可读性。

安装

访问此网站Fire code，下载.zip的文件，这里提供一个本文撰写时最新版本链接Fire Code v2.0。
版本仍在更新中，请关注Fire code以获取最新版本。

下载解压文件，里面有不同格式的字体包，选择自己合适的格式，将其复制到自己电脑的字符集之中即可完成安装。

调用

打开Sublime Text3，选择Sunlime Text -> Preferences -> Settings，打开设置页面，在"ignored_packages"之前加上"font_face": "Fira Code",，保存配置文件，重启编辑器即可生效

如果想使用字体选项，可参考

根据自己喜好，在刚刚的那一行后加入"font_options": ["ss01", "ss02", "ss03", "ss04", "ss05", "ss06", "ss19", "ss20"]

注意：目前Sublime Text不再支持zero和onum的特性，不过现在它们已经由ss19和ss20所代替。
